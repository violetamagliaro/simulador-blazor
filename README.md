# Simulador SUBE & MiniMarket 

Aplicación web desarrollada con **Blazor WebAssembly (.NET 8)** que simula procesos de negocio reales mediante dos módulos principales: un sistema de transporte tipo SUBE y un sistema de gestión de tienda.

---

## Demo en vivo

👉 https://violetamagliaro.github.io/simulador-blazor/

---

## Descripción

Este proyecto simula procesos de negocio reales aplicando lógica de dominio, reglas de negocio y programación orientada a objetos.

El objetivo es representar el funcionamiento de sistemas cotidianos como el transporte público y una tienda, implementando validaciones, estados y operaciones típicas.

---

## Simulador SUBE

Sistema de simulación de tarjeta de transporte.

### Funcionalidades:
- Carga y gestión de saldo
- Pago de viajes
- Tarifa normal y medio boleto
- Franquicias según tipo de usuario
- Registro de viajes
- Control de saldo negativo
- Validación de operaciones y restricciones

---

## MiniMarket

Sistema de simulación de tienda.

### Funcionalidades:
- Gestión de productos
- Control de stock
- Registro de ventas
- Actualización automática de inventario
- Simulación de compras

---

## Conceptos aplicados

- Programación orientada a objetos (POO)
- Modelado de dominio (Domain Modeling)
- Manejo de estado en aplicaciones web
- Lógica de negocio
- Desarrollo frontend con Blazor WebAssembly
